Projects I've Worked On

Boba Buddies

Rails, React/Redux, PostgreSQL

An app inspired by Tea With Strangers that connects strangers together for networking. It combines ActiveRecord and PostgreSQL to track users attending or hosting events through a join table. A single dashboard component is devised to render either hosting or current events for DRY code. AJAX requests are utilized to immediately update view page of React components and jQuery to retrieve the necessary data to render in JSON through jbuilders.

Live Repo

Lynk

React, React Native

A mobile app for Apple iOS platform that compares Lyft and Uber transportation costs. It uses React-Native-Maps to render pickup location and destination on map, Keyboard Avoiding View component to style input fields that adjust its position if the keyboard is active, and Geocoding API to translate location into latitude and longitude coordinates. It renders real-time ride estimations by using Lyft API and Uber API to fetch the information after submission and parsing it before being displayed.

Live Repo

Buzzin' Bee

JavaScript, Canvas, HTML5

A side-scrolling action game inspired by Jetpack Joyride. It incorporates HTML5 and Canvas to create animations and a 2D side-scrolling background. It implements Object Oriented Programming and classical inheritance for more optimized code.
